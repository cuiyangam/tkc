本项目以 [Create React App](https://github.com/facebook/create-react-app)为脚手架.



### 运行项目
1.运行 `npm start` <br>
2.访问地址 [http://localhost:3000](http://localhost:3000) <br>

### 实现功能
1.HashRouter  
2.Route  
3.Switch  
4.Redirect  
5.Link  
6.NavLink  
7.withRouter  
(HashRouter 与 BroserRouter 类似，仅以 HashRouter 为例)
